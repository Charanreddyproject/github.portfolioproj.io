Vampire attacks are a new type of resource consumption attacks that target ad hoc wireless sensor networks by depleting nodes battery power using routing protocols. Defenses against some of the forwarding-phase attacks were proposed, the first sensor network routing protocol that provably bounds damage from Vampire attacks was described. The proposed protocol verifies that packets consistently make progress toward their destinations.
